# This test hunt provided the following key takeaway points:
- Threat hunting with a SIEM-SOAR tool like Microsoft Sentinel is very useful to proactively strengthen the company's security posture.
- It enables the detection of threats that would otherwise bypass the default security sensors like Defender for Endpoint or Office (my demo subscription is also a good example for this - my test attack email did not set off an alert in Defender XDR because there is very little activity in the tenant, Defender for Office does not know what is suspicious and what is not. Without the hunting query this email would not have been detected).
